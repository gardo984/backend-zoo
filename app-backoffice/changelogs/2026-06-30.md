# Changelog — 2026-06-30

## App Backoffice (app-backoffice/)

### Project Scaffold
- Vue 3 + TypeScript via Vite
- Pinia state management
- Vue Router with auth guard
- Axios HTTP client with JWT interceptor
- Sass (SCSS) for styling
- SweetAlert2 for notifications

### Authentication
- Login UI with email + password fields
- Hardcoded password check (`123456`) for demo
- Calls `POST /login/` on FastAPI backend
- JWT token stored in localStorage, attached to all API calls via Axios interceptor

### Sidebar Navigation
- Fixed sidebar with 4 entries: Users, Categories, Authors, Books
- Active route highlighting
- User info + Sign Out button at bottom

### CRUD Maintenance Interfaces

#### Users
- List view with ID, email, disabled status
- Create new user (with password validation: min 8 chars)
- Delete user with confirmation dialog

#### Categories
- List view with ID, name, active status
- Create / Edit (inline modal) with name + active toggle
- Delete with confirmation dialog

#### Authors
- List view with ID, name, email, age, active status
- Create / Edit with name, email, age, active toggle
- Delete with confirmation dialog

#### Books
- List view with ID, name, category, author, price, active status
- Create / Edit with name, category (dropdown), author (dropdown), description, image path, price, active toggle
- Pre-loads categories and authors for select fields
- Delete with confirmation dialog

# Changelog — 2026-06-27

## Homologate Book Model (app-online-store)
- Added `category_description` (string(200), nullable) — replaces category_id concept
- Added `author_description` (string(200), nullable) — replaces author_id concept
- Added `active` (boolean, default true) — mirrors FastAPI Book.active
- Added `external_id` (unsigned int, nullable) — stores FastAPI Book ID for cross-reference
- Updated listing endpoints (admin + public) to display new fields
- **Create/update operations unchanged**
- Branch: `feature/homologate-book-model`

## Redis Publish Payload (app-library)
- Updated `router/books.py` publish calls to send full book structure instead of minimal payload
- Payload now includes: `id`, `name`, `category_description`, `author_description`, `active`, `description`, `image`, `price`
- Branch: `feature/update-redis-payload`

## Book Sync Worker (worker/)
- New NestJS project `worker/` at repo root level
- **Product entity** mirroring app-online-store (same fields)
- **RedisService** — ioredis-based subscriber, env-configurable (host/port)
- **ListenerService** — subscribes to `book_updates` Redis channel, processes incoming messages
- **ProductService** — upsert logic matching by `external_id` (FastAPI Book ID)
- Shares the same `dbstore` MariaDB database
- Built, compiles clean
- Branch: `feature/book-sync-worker`
